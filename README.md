# Weather App

A responsive weather application built with JavaScript that allows users to search for cities around the world and view current weather conditions. The app uses the OpenWeather API to retrieve live weather data and automatically updates the background based on the current weather.

## Features

- Search for weather by city
- City autocomplete suggestions
- Accurate results for cities with the same name using latitude and longitude
- Current weather conditions
- Dynamic weather backgrounds
- Fahrenheit and Celsius temperature toggle
- Welcome screen on initial launch
- Responsive glassmorphism-inspired user interface
- Displays:
  - Temperature
  - Weather description
  - Feels Like temperature
  - Wind speed
  - Humidity

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather Geocoding API
- OpenWeather Current Weather API

## Installation

1. Clone the repository.

```bash
git clone https://github.com/Rdean-dev/weather-app.git
```

2. Navigate into the project.

```bash
cd weather-app
```

3. Get a free API key from OpenWeather.

4. Replace the API key inside `index.js`.

```javascript
API_KEY = "YOUR_API_KEY";
```

5. Open `index.html` in your browser.
